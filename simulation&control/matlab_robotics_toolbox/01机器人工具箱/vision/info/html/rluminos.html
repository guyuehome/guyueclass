rluminos

Relative photopic luminosity function

p = rluminos(lambda) is the relative photopic luminosity function for the wavelengths in lambda [m]. If lambda is a vector (Nx1), then p (Nx1) is a vector whose elements are the luminosity at the corresponding elements of lambda.

Relative luminosity lies in the interval 0 to 1 which indicate the intensity with which wavelengths are perceived by the light-adapted human eye.
